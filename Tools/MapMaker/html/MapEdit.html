Editing the Map

This page will explain how you actually work with MapMaker. You should first create a new map, of size 30x30 using the Pacman tiles.

Setting tiles

Make sure that you're in Edit Mode. To do this click once at icon with the arrow in the icon bar.
Select the base layer, by clicking at the radio button labled "Base" at the right side of the window. Now, select the tile with the veritcal border by left- clicking on it. It should now be selected.
Now click once in the upper left corner of the map. Voila- you have set your first tile. Click now at the cell below, hold the button pressed, and move the mouse down. This allows you to paint with the selected tile. You can now create the whole maze by selecting the apropriate tile, and then drawing it at the correct position
Hint: You can display the cell boundaries by selecting View/ MapGrid from the menu bar.

Setting Attributes

 The next thing to do is to mark the cells as non- walkable. Switch to attribute mode by clicking on the ShowAttr button, just above the layer selecting radio buttons.
Click on one of the border cells, now use the combobox labled "Walkable area", and select "none". You could now click on the next cell, and set the attribute again, but this would take relatiyly long.

So, click on the icon, to enable "Attribute drawing". This allows you to copy all attribute at once. Just click on the cell, or move the mouse with pressed left button.
