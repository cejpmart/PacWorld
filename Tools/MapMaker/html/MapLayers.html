MapMaker allows you to use up to 4 layers. These are named: Base, Fringe, Object and Roof. The actula layer - the one at which the tiles are positioned- can by set by clicking at the coresponding button in the Layer dialog bar at the right of the window.

Hiding Specific Layers

MapMaker allows you to hide the layers. To do so, choose Layers... from the View menu, or press the "View Layers..." button at the right of the window. The now displayed Visible Layers dialog allows you to set the visibility of each layer.

Note: You can only edit visible layers.
