Map Properties

Using this dialog, which can be activated using the context menu (right mouse click on the map window) or the Edit menu, you can change the size of the map, the used tiles or the index of the transparent colors.


Changing the Map Size

Just enter the new dimensions of the map, and click the OK- button. Before decreasing the map size, MapMaker will ask you if you're really sure, since changes of the size can't be undone.


Changing the Tiles used

It's also possible to change the tiles used. Just click on the "Change..." button next to the file name of the current tiles, and select the new tile file. If this new file has less tiles then the current, all map entries with indixes higher than the tile count will be set to 0.


Setting the Transparency Index

You can change the index of the color not displayes by MapMaker using the inputline labled Trans. Index.
