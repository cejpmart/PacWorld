Troubleshooting

Problem: When I open a file, I get the message that the tile file could not be found
Your asked whether you want to search for the file- answer with "Yes", and use the file dialog to select the correct tile file.
This problem occurs if you move your tiles to a different location, or if you use maps created by other users for the first time.

Problem: When I'm drawing with tiles, sometimes they're set at the wrong position
This is bug in Public Beta 1.0. You should download a newer version.



If you have a problem not listed here, which is not answered at other positions in the documentation, please mail me.
