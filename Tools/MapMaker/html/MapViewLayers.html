Visible Layers Dialog

You can activate this dialog either using them menu item View and selecting Layers..., or by clicking the View Layers... button on the Layer Selection dialog bar at the right of the window.

All layers checked when you press the OK button will be visible, the others will be hidden. You can only set tiles to visible layers.
