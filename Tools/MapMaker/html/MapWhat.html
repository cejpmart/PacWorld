What is MapMaker ?

MapMaker is a Windows 95 program, allowing you to edit tile maps with up to 4 layers, and additional data per position. It's designed for roleplaying games, but you can create maps for jump'n'runs or sidescrolling shoot'em ups also.


Features

  • Four Tile Layers
    You can stack up to 4 tiles per position, so creating maps like those described in the TileFaq is no problem. It's also possible to hide specific layers, giving you a better overview of the map- layout.

  • Undo/ Redo for 10 steps

  • Up to 65535 tiles per map
    This should be sufficient for even large projects.

  • Multi Document Interface (MDI)
    MapMaker allows you to open several maps at the same and to copy and paste data between them. Since it follows the standard Windows guidelines, you can start to use the program without having to learn the user interface, first.

  • Two Types of Tile Windows
    If the standard tile window gets to small for your tiles, just view them in the multi- coloumn tile window.

  • Friendly Programmer
    The best feature of MapMaker is, that if you have ideas for improvement, all you have to do, is to mail me. If your idea is implementable, it will be implemented.
